## ğŸ¨ Customization

### Colors

Edit the CSS variables in `assets/css/style.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... more variables */
}
```

### Content

Edit the content directly in `index.html` - all text, sections, and links are easily customizable.

### Images

The website currently uses Unsplash placeholder images. To use your own images:

1. Add your images to `assets/images/` folder
2. Update the `src` attributes in `index.html` to point to your local images

See `assets/images/README.md` for image requirements.
